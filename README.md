# Project: E-commerce Website

## Installation
1. Clone the project
2. Install yarn
    ```bash
    npm install -g yarn
    ```
3. Install dependencies
    ```bash
    yarn install
    ```
4. Create a `.env` file in the root directory and add the following environment variables (take a look at env.example)
5. Start the server
    ```bash
    yarn start
    ```
